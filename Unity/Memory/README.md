
## üîÑ Memory Exchange Protocol

### **Version 2.0 Workflow**

When completing work, create a **memory snapshot**:

```bash
# Navigate to eMemory directory
cd /Users/mdhowell/eestream/eMemory

# Create versioned ZIP snapshot
zip -r eMemory_2025-11-15_1630_CLERK_v2.zip \
  01_Context/ 02_Systems/ 03_Knowledge/ 04_Conversations/ 05_Logs/ \
  README.md METADATA_TEMPLATE.md
```

### **Naming Convention**

```
eMemory_YYYY-MM-DD_HHMM_[AGENT]_v[VERSION].zip

Examples:
  eMemory_2025-11-15_1630_CLERK_v2.zip
  eMemory_2025-11-15_1845_CODEX_v1.zip
  eMemory_2025-11-16_0900_COVE_v3.zip
```

**Components**:
- **Date/Time**: When snapshot was created (UTC or local timezone)
- **Agent**: CLERK, CODEX, or COVE (who created this version)
- **Version**: Incremental version number for that day/session
