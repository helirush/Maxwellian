## Core Project Identity
Unity Energy's eestream system is an advanced software platform that reveals the hidden behavior of industrial energy fields through patented MPTS technology. The mission is to educate and empower industry leaders to understand and steward electricity at the 480V industrial level.

## Primary Objectives
1. **Measure, Manage, Maximize** - Complete industrial energy analysis workflow
2. **Reclaim Reactive Energy** - Advanced energy field analysis and optimization
3. **Redefine Industrial Electricity** - Revolutionary approach to energy stewardship
4. **Comprehensive Analysis Suite** - Four core modules plus unified dashboard generation

## Core System Architecture

### Four Primary Modules
1. **eBehavior** - Advanced Energy Analysis & Intelligence (Main Tool: `eeBEHAVIOR.py`)
2. **eVision** - Visual Data Interface & Real-Time Analysis (Streamlit-based: `eeVISION.py`)
3. **eAudio** - Voice & Audio Processing (OpenAI TTS + PyAudio)
4. **eMarket** - Market Analysis & Economics

### Major 2025 Features
- **Unified Dashboard System** - Three dashboard types (Energy, Heat, Voltage)
- **Advanced Voltage Analysis** - Critical warnings system with G8 motor calculations
- **Enhanced Reporting** - Transformer-centric organization with markdown generation
- **Directory Structure Management** - Automated facility → site → transformer organization

## Key Technical Components
- **Data Processing Pipeline**: Raw CSV → ePrep → eWeather → eBehavior → E-Stream Ready
- **Dashboard Templates**: Energy, Heat, and Voltage dashboards with live data integration
- **Voltage Health Analysis**: Advanced forensics with critical warnings system
- **Thermal Analysis**: Heat model calculations and cooling load assessments
- **Financial Analysis**: Utility billing integration with quarterly comparisons

## Success Metrics
- Complete workflow automation from raw data to interactive dashboards
- Transformer-centric report organization and management
- Advanced voltage quality assessment and early warning systems
- Economic analysis and ROI calculations for energy optimization
- Professional-grade technical documentation and reporting

## Technology Stack
- **Core**: Python 3.10+, Streamlit for visual interface
- **AI Integration**: OpenAI GPT-4o for analysis and TTS
- **Voice Processing**: ElevenLabs integration, PyAudio streaming
- **Data Management**: CSV processing, SQL integration, automated organization
- **Output Formats**: Interactive HTML dashboards, Markdown reports, data exports

This system represents the future of industrial energy analysis, providing unprecedented insight into electrical energy behavior at the industrial level.
